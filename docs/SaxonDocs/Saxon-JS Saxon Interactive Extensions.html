Saxon Interactive Extensions

Interactive XSLT extensions in the http://saxonica.com/ns/interactiveXSLT namespace were introduced by Saxon for use with Saxon-CE in the browser. The same interactive XSLT is used by Saxon-JS in the browser, with some new additions.

The standard prefix used for the namespace http://saxonica.com/ns/interactiveXSLT is ixsl:.

This section of the documentation contains a reference library for the interactive XSLT elements and functions that may be used.
