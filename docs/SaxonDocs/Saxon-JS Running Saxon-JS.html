Running Saxon-JS

Saxon-JS is loaded using a standard JavaScript script element within the HTML head or body:

<script src="SaxonJS/SaxonJS.js"></script>

or:

<script src="SaxonJS/SaxonJS.min.js"></script>

To execute a transform using Saxon-JS, use the transform method of the SaxonJS object. For example:

<script> window.onload = function() { SaxonJS.transform({ stylesheetLocation: "xsl/mytrans.sef", sourceLocation: "source.xml" }); } </script>

The first argument for the transform method is a JavaScript literal object supplying the transform options (much like the XPath 3.1 transform() function). For details about the options that can be supplied see SaxonJS.transform.
