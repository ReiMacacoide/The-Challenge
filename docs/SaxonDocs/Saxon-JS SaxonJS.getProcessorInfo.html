SaxonJS.getProcessorInfo

SaxonJS.getProcessorInfo()

The method takes no arguments. Returns an object containing standard information about the processor:

  • version
  • vendor
  • vendorURL
  • productName
  • productVersion
  • isSchemaAware
  • supportsSerialization
  • supportsBackwardsCompatibility
  • supportsNamespaceAxis
  • supportsStreaming
  • supportsDynamicEvaluation
  • supportsHigherOrderFunctions
  • xPathVersion
  • xsdVersion
