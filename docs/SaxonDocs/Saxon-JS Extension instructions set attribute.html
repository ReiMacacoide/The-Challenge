ixsl:set-attribute

Sets an attribute with a given name and value on the current element node. The current node must be an element in the HTML page.

Requires Saxon-EE. Implemented since Saxon 9.7.0.7.

Category: instruction
Content: sequence-constructor
Permitted parent elements: any XSLT element whose content model is sequence-constructor; any literal result element

Attributes

name

{ eqname }

The name of the attribute to be added, interpreted as an attribute value template, so it may contain string expressions within curly braces.

select

expression

The attribute value.

Details

The instruction has attributes name and select, and works in exactly the same way as xsl:attribute, except that the attribute is written directly to the current element. The instruction follows the same restrictions as xsl:result-document in that it should not be used while evaluating a variable or a function.

An error will be raised at run-time if the current context item is not an element in the HTML page.

Examples

Set the class attribute on an HTML element:

<ixsl:set-attribute name="class" select="'closed'"/>
