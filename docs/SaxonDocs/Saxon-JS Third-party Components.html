Third-party Components

Big

big.js v3.1.3 is a library for handling arbitrary-precision decimal arithmetic. It is used by Saxon to handle xs:decimal values, and xs:integer values that are too large to accommodate in the JavaScript number data type.

big.js is Copyright (c) 2014 Michael Mclaughlin (M8ch88l@gmail.com) and is made available under the MIT Expat license. The version used by Saxon-JS is version 3.1.3, downloaded from https://github.com/MikeMcl/big.js/ on 2016-01-11.

XML Parsing Libraries

When running on the browser, Saxon-JS uses the XML parsing libraries provided by the browser platform.

On server environments (Nashorn and Node.js), Saxon-JS uses the following libraries:

  • xmldom from https://github.com/jindw/xmldom. This package provides a SAX parser, DOM tree builder, and DOM tree implementation.
  • sax-js from https://github.com/isaacs/sax-js. This package provides a SAX parser only. It appears to do better XML error checking than the parser in the xmldom package, so Saxon currently uses both.
