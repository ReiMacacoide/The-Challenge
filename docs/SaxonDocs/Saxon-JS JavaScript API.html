JavaScript API

This section describes the JavaScript API for Saxon-JS. Once the Saxon-JS library is loaded, the principal object SaxonJS is available.

The SaxonJS object provides four main utility functions.

Function Description

transform

The main method used to invoke a transformation by using a JavaScript literal object.

getProcessorInfo

Returns an object containing standard information about the processor.

setLogLevel

Requires Saxon-JS 1.1.0 or later.

Sets the logging level for warnings, processing messages, or full tracing.

XPath.evaluate

Dynamic evaluation of an XPath 3.1 expression.

Support for XPath.evaluate involves the dynamic loading of code from the JavaScript package lib/xpath/XPathJS.min.js, which is performed automatically on the first call.
