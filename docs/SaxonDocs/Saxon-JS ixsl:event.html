ixsl:event

Returns the current Event object while processing a user input event (or the empty sequence otherwise).

Requires Saxon-EE. Implemented since Saxon 9.7.0.7.

event() ➔ item()?

There are no arguments

Result

item()?

Namespace

http://saxonica.com/ns/interactiveXSLT

Notes on the Saxon implementation

Available since Saxon 9.7.0.7.

Details

The returned Event is a JavaScript object: it is possible to get its properties using ixsl:get() and to call its methods using ixsl:call().

If there is no current Event, then the function returns the empty sequence.

Examples

Event handling template to handle a keydown KeyboardEvent, using ixsl:event() to obtain the key which has been pressed:

<xsl:template match="." mode="ixsl:onkeydown"> <xsl:variable select="ixsl:get(ixsl:event(), 'key')" /> ... </xsl:template>
