Licensing

Although the source code of Saxon-JS is made available, the product is not open source. The code is the intellectual property of Saxonica, except for open source components listed below.

For licensing purposes Saxon-JS is considered to be part of Saxon Enterprise Edition, and purchasing a Saxon-EE license entitles you to use and deploy Saxon-JS. The basic model for Saxon-JS is that we charge for development, not for deployment. A Saxon-EE license is required to produce an SEF (stylesheet export file), but no license is needed for run time deployment on a web browser. The Saxon-EE license conditions can be found at Saxon Licensing.

You are permitted to make modifications to the code for your own internal use, for example for debugging and fixing bugs, for the resolution of performance or security problems, for localization, or to get the product to work in environments where Saxonica has not tested it. The one condition is that you must hand over any such modifications to Saxonica to include in the product at Saxonica's discretion.
